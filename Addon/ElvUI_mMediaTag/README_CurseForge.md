# mMediaTag

`mMediaTag` is an ElvUI plugin for WoW Retail/Mainline that expands ElvUI with custom portraits, dock-style datatext icons, utility datatexts, custom tags, nameplate helpers, minimap and panel skins, and several quality-of-life modules.

Version `4.x` is the Midnight-ready branch and is designed for Retail only.

## Features

- Custom portrait system with many bundled styles, shadows, overlays, class icons, spec icons, and custom texture support
- Dock icon modules for character, friends, guild, LFG, spellbook, talents, professions, mail, quests, bags, achievements, calendar, volume, encounter journal, and more
- Extra datatexts for Mythic+ score, teleports, professions, game menu, dungeon info, combat time, durability and item level, and trackers
- Custom ElvUI tags for classification, health, status, role, level, faction, PvP, quests, class icons, and death counters
- Nameplate tools such as target glow color syncing and highlight helpers
- Utility modules including auto quest, auto role check, tooltip tweaks, greeting message, LFG invite info, Details embedding, minimap skin, and panel skin options

## Requirements

- WoW Retail / Mainline
- ElvUI

Optional:

- `Details`
- `ElvUI_JiberishIcons`

## How To Access Settings

Open:

```text
ElvUI > mMT
```

The addon is fully integrated into the ElvUI config.

## Slash Commands

- `/mmt`
- `/mmt help`
- `/mmt version`

## Notes

- Retail/Mainline only
- `4.x` is the current Midnight-oriented branch
- This is not the Classic branch
- Includes localization files for multiple languages
